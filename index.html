(1) Basic HTML document

<!DOCTYPE html> <html>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>
  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document (tells to the Browser, what is the version of HTML
  • The <html> element is the root element of an HTML page tells to the Browser that everything between the opening, and closing tag will be an HTML- code).
  • The <head> element contains meta information about the HTML page.
  • The <title> element specifies a title for the HTML page (which are shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading and <h6> element defines a smallest heading.
  • The <p> element defines a paragraph

(2) How to do comment

<!-- This is a comment --> or
<!-- This is a comment too -->
Comments are portion of the code that have been marked out to be ignored by the browser.

(3) HTML <a> Tag

<a href="https://www.w3schools.com">Visit W3Schools.com!</a>
  • The <a> tag defines a hyperlink, which is used to link from one page to another.

(6) <img>: The Image Embed element

(7) HTML Tables

<table style="width:100%">
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
</table>

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
  • The <table> tag defines an HTML table.
  • Each table row is defined with a <tr> tag. Each table header is defined with a <th> tag. Each table data/cell is defined with a tag.
  • By default, the text in <th> elements are bold and centered.
  • By default, the text in <td> elements are regular and left-aligned.
